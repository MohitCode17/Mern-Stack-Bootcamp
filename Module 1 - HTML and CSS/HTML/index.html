Image Galley

Images play a very important role in HTML and web development because they make web pages more engaging, visually appealing, and informative.

Image tag is self closing tag which contain some attributes(Tells more informtaion about tag) like: src is a source of image file, alt(Important for SEO and accessibility) used for alternate text used when image loads to failed.

HTML Links

Used when you need to link another documents within a HTML file. Contain attributes like href tells the destination of link and target tells the where to open documents.
